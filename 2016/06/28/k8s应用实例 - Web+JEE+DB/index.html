发表于 2016-06-28 更新于 2016-08-09 分类于 k8s应用实例

系统架构:

nginx+tomcat+mysql

部署规划:

  • mysql采用单个Pod，tomcat、nginx分别用rc来部署。
  • 持久化数据采用Persistent Volumes挂载NFS来支持。

前期准备:

访问方式: http://NodeIP:NodePort

下来就可以部署我们的应用系统了：

  • JEE应用：将.war包放到tomcat的webapps目录，这里放到NFS服务器的对应目录即可 /data/tomcat/
  • Web应用：将站点文件放到nginx相应的配置路径，这里存放到NFS服务器对应目录 /data/web
  • DB：我们前面已经手动恢复过。

此时应用已经可以正常访问了，http://NodeIP:NodePort
