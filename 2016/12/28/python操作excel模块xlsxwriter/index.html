XlsxWriter介绍

XlsxWriter是一个用来创建Excel XLSX文件的python模块。
XlsxWriter官网地址: http://xlsxwriter.readthedocs.io/

下面一个小例子展示XlsxWriter模块的一些特性：




功能列表

XlsxWriter具有以下功能：

  • 100%兼容Excel XLSX文件，支持Excel2007及已上所有版本
  • 支持所有Excel单元格数据格式
  • 支持单元格合并、批注、自动筛选、丰富多格式字符串等
  • 支持工作表PNG/JPEG图像，自定义图像
  • 内存优化模式支持写入大文件

模块常用方法说明

Workbook类

Workbook类定义：Workbook(filename[, options])
创建一个XlsxWriter的Workbook对象。Workbook代表整个Excel文件，并存储在磁盘上。
参数:

  • filename(string) Excel:创建的文件的存储路径。
  • optins(dict): 可选的workbook参数，一般用来初始化工作表的内容格式，具体官网上有详细的书面。

workbook.add_worksheet([sheetname])方法

在workbook里添加一个新的工作表。
参数:

  • sheetname(string):可选工作表名称，默认为sheet1。

workbook.add_format([properties])方法

在工作表中创建一个新的格式对象来格式化单元格。

Worksheet类

Worksheet类代表了一个Excel工作表，是XlsxWriter模块最核心的一个类，例如将数据写入单元格或工作表布局等。
Worksheet对象不能直接实例化，需要通过Workbook()对象使用add_worksheet()方法来创建。
简例:

worksheet.write(row, col, *args)方法

将普通数据写入工作表的单元格。
参数:

  • row:行坐标，起始为0
  • col:列坐标，起始为0
  • *args:数据内容，可以为字符串、数字、公式或格式对象。

为了简化不同类型数据的写入，write方法提供其他更为具体数据类型的方法：

  • write_string() 字符串
  • write_number() 数字
  • write_blank() 空类型
  • write_formula() 公式类型
  • write_datetime() 日期类型
  • write_boolean() 逻辑类型
  • write_url() 超链接类型

worksheet.set_row(row,height,cell_format,options)方法

设置行单元格属性。
参数:

  • row(int): 指定行位置，起始下标为0
  • height(float): 行高，单位是像素
  • cell_format(Format): 指定格式对象
  • options(dict): 设置行参数，hidden(隐藏),level(组合分级),collapsed(折叠)等

示例: 设置第二行高度15像素，应用加粗格式并隐藏

worksheet.set_column(first_col,last_col,width,cell_format,options)方法

设置一列或多列单元格属性。
参数:

  • first_col: 指定开始列位置，起始位0
  • last_col: 指定结束列位置，起始位0，可与first_col相同
  • width: 设置列宽，单位为像素
  • cell_format: 指定格式对象
  • options: 设置列参数，hidden(隐藏),level(组合分级),collapsed(折叠)等
