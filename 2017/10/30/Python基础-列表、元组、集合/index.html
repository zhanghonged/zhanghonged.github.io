Python基础-列表、元组、集合

发表于 2017-10-30 更新于 2017-11-12 分类于 python

列表

列表是一个有序的，可修改的，元素以逗号分割，以中括号包围的序列。

列表的定义

1、[]
2、list
3、range
4、xrange

列表的更新

可操作列表的函数

  • cmp 比较两个列表的元素，一样返回0，如果第一个参数的列表大于第二个参数的列表，返回1，否则返回-1
  • len 获取列表的长度
  • max 返回列表当中的最大值
  • min 返回列表当中的最小值
  • append 在列表末尾添加新的元素
  • count 统计某个元素在列表中出现的次数
  • extend 在列表末尾一次性追加一个序列中的多个值(用于新列表的拓展)
  • index 在列表中从左到右找出某个值的第一个匹配项的索引位置
  • insert 将对象插入列表当中
  • pop 删除列表当中的最后一个元素，并返回该元素的值
  • remove 删除列表中从左到右第一个匹配到的值
  • reserve 反向列表中的元素
  • sort 对列表进行排序(默认从小到大)，可以加reverse=True从大到小进行排序

元组

元组是元素以逗号分隔，以小括号包围的有序的序列，是不可修改的序列。

元组的特性

3、元组不可修改，所以配置文件中用的较多

元组和字符串的区别

  • 元组和字符串都是有序的，不可被修改的序列
  • 元组的元素可以是任何元素，但字符串的元素只能是字符
  • 元组的元素长度可以任意，而字符串的元素长度只能为1

元组的方法

  • index 从左到右开始查找，返回第一个匹配到的指定元素的索引值，如果没用，报错
  • count 返回元组当中指定元素的个数

集合

集合有两种类型

  • set 可变集合,可以添加和删除元素
  • frozenset 不可变集合，不可以添加和删除元素

创建集合

set() 可变集合
frozenset() 不可变集合

集合的特性

  1. 集合没有索引的概念
  2. 集合不可被修改
  3. 集合是一个无序序列
  4. 集合去重

访问集合中的元素
集合没有索引的概念，所以得用遍历的方法来查看集合中的元素

更新集合
