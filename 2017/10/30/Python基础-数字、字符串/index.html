Python基础-数字、字符串

发表于 2017-10-30 更新于 2017-11-12 分类于 python

数字

在python当中使用比较广泛的数据类型

数字类型

  • 整形： int 1
  • 浮点型： float 1.0
  • 长整型： long 当一个整数后面有L，那么这个数值就是长整型，Python2当中超过10位视为长整型

type() 返回表达式的值的类型

数字转换

1、通过类型转换

2、通过行为转换

数字运算

  • +
  • /
  • % 取余，不能整除，取余数
  • ** 求方
  • // 取整除 - 返回商的整数部分

字符串

字符串是一个有序的,不可修改的,元素以引号包围的序列(由数字,字母,下划线组成)

字符串定义

  • ‘string
  • “string”
  • ‘’’string’’’
  • “””string”””

str() 类型函数

单引号和双引号主要用来区分

三引号（三个单引号和三个双引号）和非三引号（单引号和双引号）区别在于三引号可以形成带有换行的多行字符串
我们通常会用三引号作为长注释，多行注释。

字符串的索引(index)

在python当中所有的有序序列都是有索引概念的，但是区别在于序列可不可以被修改。
索引在我们初学的时候可以理解为字符串的下标
字符串的每一个个体都被称作为该字符串的一个元素

索引取值

  • 从左到右索引默认从0开始，最大范围是字符串长度减1
  • 从右到左索引从-1开始，最大范围是字符串开头

获取字符串的长度: len()

索引用法

  • 取单个元素:字符串(变量)[索引值] 对应索引值的元素
  • 截取:字符串[start:end] 得到对应范围的元素，该范围包含起始端，不包含结尾端
  • 步长截取字符串[start:end:step] 按照步长减1进行截取
  • 默认取法:字符串[start:end:step] 这三个参数都有默认值。(start 0、end 结尾、step 1)
  • 反取:字符串[负数] 从右往左开始取

字符串方法

字符串的修饰

  • center 让字符串在指定的长度居中，如果不能居中左短右长，可以指定填充内容，默认是以空格来填充
  • ljust 让字符串在指定的长度左齐，可以指定填充内容，默认是以空格来填充
  • rjust 让字符串在指定的长度右齐，可以指定填充内容，默认是以空格来填充
  • zfill 将字符串填充到指定的长度，不足地方用0从左开始补充
  • format 按照顺序，将后面的参数传递给前面的大括号
  • strip 默认去除两边的空格，去除内容可以指定
  • rstrip 默认去除右边的空格，去除内容可以指定
  • lstrip 默认去除左边的空格，去除内容可以指定

字符串的查找

  • count 计数功能，返回指定字符串在字符串当中的个数
  • find 从左开始查找，返回匹配到的第一个索引值，找不到返回-1
  • rfind 从右开始查找，返回匹配到的第一个索引值，找不到返回-1
  • index 从左开始查找，返回匹配到的第一个索引值，找不到报错
  • rindex 从右开始查找，返回匹配到的第一个索引值，找不到报错

字符串的替换

  • replace 从左到右替换指定的元素，可以指定替换的个数，默认全部替换
  • translate 按照对应关系来替换内容 from string import maketrans

字符串的变形

  • upper 将字符串当中的所有字母转换为大写
  • lower 将字符串当中的所有字母转换为小写
  • swapcase 将字符串当中的所有字母大小写进行互换
  • title 将字符串当中的所有单词首字母大写然后非首字母小写，单词以非字母划分
  • capitalize 将整个字符串当中的首字母大写（只有首字母），其他字母会被转换为小写，如果开头是非字母，则不会转换
  • expandtabs 修改字符串当中\t的长度

字符串的判断

  • isalnum 判断字符串是否完全由字母和数字组成
  • isalpha 判断字符串是否完全由字母组成
  • isdigit 判断字符串是否完全由数字组成
  • isupper 判断字符串当中的字母是否完全是大写
  • islower 判断字符串当中的字母是否完全是小写
  • istitile 判断字符串是否满足title格式
  • isspace 判断字符串是否完全由空格组成
  • startswith 判断字符串开头字符是否为给定的元素开头，可以截取判断
  • endswith 判断字符串是否以指定字符或子字符串结尾，常用于判断文件类型，可以截取判断

字符串的切分

  • splitlines 以行切分字符串
  • split 从左开始切分字符串，默认以空格切分，可以指定切分的次数
  • rsplit 从右开始切分字符串，默认以空格切分，可以指定切分的次数

字符串的拼接

  • join 将指定字符串插入到后面的序列的每两个元素之间进行拼接，形成一个新的字符串
  • “+” 将两个字符串拼接起来
  • ”*“ 将指定的字符串进行重复

字符串的转义

比如\n，\t，\，本身都需要转义，所以前面加一个’\’表示转义
如果字符串当中有很多字符需要转义，可以再字符串的前面加r’’，表示原始字符串。
