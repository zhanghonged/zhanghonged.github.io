Python基础-字典

发表于 2017-11-01 | 分类于 python

字典
字典是元素以键值对形式存在，以逗号分割，以大括号包围的无序的，可以修改的序列。
字典是python基础数据类型当中唯一一个映射关系的数据类型，通常对应JSON。

字典的定义

zip 函数：将几个序列对应索引位上的元素分到一个元组当中，形成一个列表，子元组的个数取决于最短序列的长度。

dict 函数

用于创建一个字典。

{}

fromkeys

用于创建一个新字典，以序列中元素做字典的键，value为字典所有键对应的初始值,默认的值为None，可以指定值。

字典的特性

  1. 因为字典是无序的，所以字典没有索引值
  2. 因为字典是没有索引值，所以字典以键取值
  3. 因为字典以键取值，所以字典的键唯一并且不可修改
  4. 因为字典的键不可修改，所以列表和字典不可以给字典做键

字典的方法

  • keys 获取所有的键
  • values 获取所有的值
  • item 返回一个列表，字典中所有键值呈元组形式的格式
  • get 以键取值，如果键不存在，默认返回None，可以指定返回值
  • update 更新指定键的内容，如果键不存在，默认创建
  • setdefault 设置默认，如果键存在，返回值，如果键不存在，返回None，并创建这个键，值默认为None，也可以指定值
  • pop 删除指定键对应的值，并返回该值
  • popitem 随机删除一个键值对元组，并返回该键值对
  • clear 清空字典
  • has_key 判断指定的键是否存在于字典当中，如果存在返回True，否则返回False，即将废除，之后我们多用in判断
  • iteritems 返回值不是列表，而是一个迭代器。如果我们想要获取具体的值，需要借助next()方法
  • iterkeys
  • itervalues

深浅拷贝

copy.copy浅拷贝: 只拷贝顶级对象，或者说是父对象。
copy.deepcopy深拷贝: 拷贝所有对象，包括顶级对象及嵌套对象，或者说父对象和子对象
