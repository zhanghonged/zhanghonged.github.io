python 函数编程，面向过程编程

函数的定义

代码块：就是将我们要执行的代码进行结构的整合，形成可被调用的代码块。
函数是可以重复调用的。
函数比较代码复制:

  1. 减少冗余代码
  2. 代码结构更加清晰
  3. 有助于保持代码的一致性

定义函数语法

1、 关键字：def
2、 函数名命名规范

  1. 字母开头
  2. 不允许有关键字
  3. 不允许有特殊符号
  4. 不建议起莫名其妙的函数名(A,B)

3、参数：参数是定义在括号里边，由调用的时候传入，作用在函数内部的变量

  1. 如果有参数，写在括号里边
  2. 如果没有，那就写空括号

4、 内容块

函数的调用

要调用一个函数，需要知道函数的名称和参数，函数在没有调用之前是不会执行的。
调用：函数名()，并且进行对应的传参形式。
实参和形参:
形参：在我们定义函数的时候定义的参数
实参：在我们调用的时候传递的值

函数的划分

按照参数类型划分：

  • 位置参数
  • 默认参数
  • 可变参数
  • 关键字参数

1、位置参数

是我们在传参的时候，实参传递的顺序是按照形参定义的顺序进行传递的传参方式。

power(x, n)函数有两个参数：x和n，这两个参数都是位置参数，调用函数时，传入的两个值按照位置顺序依次赋给参数x和n。

2、默认参数(缺省参数)

是在我们定义参数的时候，我们给形参一个默认值，在我们调用的时候，如果不给有默认值的形参参数，会自动采用默认值。
注：默认值参数必须放在正常参数的后面，否则会引发语法错误。

return 函数退出，返回一个值给调用方，不带值的return相当于返回None。
