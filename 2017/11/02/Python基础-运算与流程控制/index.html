逻辑运算

“且” and 只有两边为真才为真，必须是两边都成立

“或” or 两边有一边为真即为真

“**非” not

逻辑运算的优先级：

  • 非的优先级最高
  • 或 且的优先级
    • 且的优先级高于或

非(not) > 且(and) > 或(or)

成员运算

测试实例中是否包含一系列的成员

  • in 如果在指定的序列中找到值返回True，否则返回False
  • not in 如果在指定的序列中找不到值返回True，否则返回False

身份运算

用于比较两个对象的存储单元(内存地址)

  • is 是判断两个标识符是不是引用自同一个对象
  • is not 是判断两个标识符是不是引用自不同对象

is和**==**的区别：
is判断的是不是引用自同一个id(),而 == 只是判断两个值是否相等

流程控制

Python当中代码正常情况下执行的顺序是从左到右，从上到下，但是在流程控制当中，并不是这样的。

if语句

if 判断 只有满足if条件的时候，才会执行if下面的代码块
if 条件:
　　blcok
else:
　　blcok

if elif else
if 条件1:
　　blcok
elif 条件2:
　　block
else:
　　block

for循环

Python中采用的是for in 结构,迭代器原理。
格式：
　　for 变量 in 可迭代的对象（序列）:
　　　　block
for循环，实质上是将后面可迭代对象的元素依次赋值给前面的变量，每赋值一次，执行一次代码块，也就是说循环一次。

特殊for循环

enumerate枚举

指出序列的索引,可以指定索引从几开始，默认为0

while

用于循环执行程序，在某种条件下，循环执行某段程序，以处理需要重复的相同代码
格式：
while 条件:
　　#判断条件可以是任何表达式，任何非0，非空的值
　　block
注：如果条件为真，while不会中断循环，我们需要定义一个条件，防止死循环。

continue、break、pass
Python continue 语句跳出本次循环，而break跳出整个循环。
continue 语句用来告诉Python跳过当前循环的剩余语句，然后继续进行下一轮循环。
continue语句用在while和for循环中。

Python pass是空语句，是为了保持程序结构的完整性。
passass 不做任何事情，一般用做占位语句。
