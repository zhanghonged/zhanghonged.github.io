Python基础-time、datetime模块

发表于 2017-11-06 更新于 2017-11-07 分类于 python

&emsp;Python具有良好的时间和日期管理功能。实际上，计算机只会维护一个挂钟时间(wall clock time)，这个时间是从某个固定时间起点到现在的时间间隔。时间起点的选择与计算机相关，但一台计算机的话，这一时间起点是固定的。其它的日期信息都是从这一时间计算得到的。此外，计算机还可以测量CPU实际上运行的时间，也就是处理器时间(processor clock time)，以测量计算机性能。当CPU处于闲置状态时，处理器时间会暂停。

time模块

时区
UTC（Coordinated Universal Time，世界协调时）亦即格林威治天文时间，世界标准时间。在中国为UTC+8。DST（Daylight Saving Time）即夏令时。

时间戳
距离1970年1月1日0:00过了多少秒。

时间格式

1、字符串时间
2、时间戳
3、元组方式(共9个元素)，也叫九元素时间
元组（struct_time）方式：struct_time元组共有9个元素，返回struct_time的函数主要有gmtime()，localtime()，strptime()。下面列出这种方式元组中的几个元素：

索引(index) 属性(Attribute) 值(values)
0 tm_year（年） 比如2011
1 tm_mon（月） 1 - 12
2 tm_mday（日） 1 - 31
3 tm_hour（时） 0 - 23
4 tm_min（分） 0 - 59
5 tm_sec（秒） 0 - 61
6 tm_wday（weekday） 0 - 6（0表示周日）
7 tm_yday（一年中的第几天） 1 - 366
8 tm_isdst（是否是夏令时） 默认为-1

time方法

1、time.localtime([secs])

将一个时间戳转换为当前时区的struct_time。secs参数未提供，则以当前时间为准。

2、time.gmtime([secs])

和localtime()方法类似,返回0时区的struct_time(九元素时间)。

3、time.time()

返回当前时间的时间戳。

4、time.mktime(t)

将一个struct_time转化为时间戳。

5、time.asctime([t])

把一个表示时间的元组或者struct_time表示为字符串时间样式，比如：’Sun Jun 20 23:21:05 1993’。如果没有参数，将会将time.localtime()作为参数传入。

6、time.ctime([secs])

将时间戳转换为字符串时间,如果没有参数，将会将time.localtime()作为参数传入。

7、time.strftime(format[, t])

把一个代表时间的元组或者struct_time（如由time.localtime()和time.gmtime()）转化为格式化的时间字符串。 如果t未指定，将传入time.localtime()。如果元组中任何一个元素越界，ValueError的错误将会被抛出。

strftime格式符

格式 含义
%Y 年(四位)
%y 年(二位)
%m 月
%d 日
%H 时(24小时制)
%I 时(24小时制)
%M 分
%S 秒

8、time.strptime(string[, format])

根据指定的时间把一个字符串时间转换为时间元组，实际上它和strftime()是逆操作。

9、time.sleep(secs)

就是让程序挂起指定的秒数。

10、time.clock

在Linux中，他返回的是进程时间，是用秒表示时间戳(浮点数)，在win中，第一次调用返回的是进程运行的实际时间，第二次调用时自第一次调用之后到现在运行的时间。

总结

在Python中共有三种表达方式：1、timestamp 2、tuple或者struct_time 3、格式化字符串。

datetime模块

datetime包是基于time包的一个高级包，是Python处理日期和时间的标准库。

介绍

datetime定义了两个常量：

  • datetime.MINYEAR (最小年份1)
  • datetime.MAXYEAR (最大年份9999)

datetime模块定义了下面这几个类：

  • datetime.date：表示日期的类(相当于日历)。常用的属性有year, month, day
  • datetime.time：表示时间的类(相当于手表)。常用的属性有hour, minute, second, microsecond
  • datetime.datetime：表示日期时间
  • datetime.timedelta：表示时间间隔，即两个时间点之间的长度
  • datetime.tzinfo：与时区有关的相关信息

运算timedelta

datetime包还定义了时间间隔对象(timedelta)。一个时间点(datetime)加上一个时间间隔(timedelta)可以得到一个新的时间点(datetime)。同理，两个时间点相减会得到一个时间间隔。
只有**”datetime.datetime”**类型的数据才能进行datetime.timedelta运算。

在给datetime.timedelta传递参数（如上的seconds和weeks）的时候，还可以是days, hours, milliseconds, microseconds。

datetime对象与字符串转换

我们通过format来告知Python我们的str字符串中包含的日期的格式。在format中，%Y表示年所出现的位置, %m表示月份所出现的位置……。
反过来，我们也可以调用datetime对象的strftime()方法，来将datetime对象转换为特定格式的字符串。

datetime对象与时间戳转换

要把timestamp转换为datetime，使用datetime提供的fromtimestamp()方法。
timestamp是一个浮点数，它没有时区的概念，而datetime是有时区的。默认转换是在timestamp和本地时间做转换。
