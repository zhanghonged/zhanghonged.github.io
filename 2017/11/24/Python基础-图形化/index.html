Python GUI介绍

.pyw 是python中图形化脚本的专属后缀，可以直接双击打开。

python图形化有多种库支持

  • Tkinter python内置的标准GUI库
  • wxpython 跨平台的图形化库
  • pywin 基于win的图形化
  • pyqt4 基于qt的跨平台图形化

GUI原理上就是一个主循环(死循环)，主体程序就是在一个循环里，关闭界面后循环结束。
主要包含的内容：
事件： 键盘事件、鼠标单击、鼠标滑过
组件： 滚动条、窗口、按钮、文本框

wx基本组件

wx的组件都是对象(都可以继承重写)

Frame 窗口

  • parent 指向的父组件，主窗口的parent为None
  • id 是组件的唯一标识，指定id为-1，即为系统自动分配ID
  • title 标题，窗口上方显示的内容
  • pos 坐标，距离左上角的坐标
  • size 组件的大小，如果窗口只有一个组件，size无效
  • style 组件的样式
  • name 和id一样，用来标识组件

TextCtrl 文本框

  • parent 指向的父组件，主窗口的parent为None
  • id 是组件的唯一标识，指定id为-1，即为系统自动分配ID
  • value 文本框显示的内容
  • pos 坐标，距离左上角的坐标
  • size 组件的大小，如果窗口只有一个组件，size无效
  • style 组件的样式
  • name 和id一样，用来标识组件

Button 按钮

  • parent 指向的父组件，主窗口的parent为None
  • id 是组件的唯一标识，指定id为-1，即为系统自动分配ID
  • label 按钮上显示的内容
  • pos 坐标，距离左上角的坐标
  • size 组件的大小，如果窗口只有一个组件，size无效
  • style 组件的样式
  • name 和id一样，用来标识组件

尺寸器

尺寸器是按照比例布局的

  1. 尺寸器不是放在窗口组件上，而是放在画布(panel)上
  2. 尺寸器水平尺寸器(默认的)和垂直尺寸器(wx.VERTICAL)
  3. 同一个画布上面可以有多个尺寸器，但是必须有一个主尺寸器
