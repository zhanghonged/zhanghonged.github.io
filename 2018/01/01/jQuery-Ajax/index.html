Ajax概述

&emsp;&emsp;Ajax 全称为：“Asynchronous JavaScript and XML”（异步 JavaScript 和 XML），它并不是 JavaScript 的一种单一技术，而是利用了一系列交互式网页应用相关的技术所形成的结合体。使用 Ajax，我们可以无刷新状态更新页面，并且实现异步提交，提升了用户体验。

Ajax并不是单一的技术，而是一串技术的集合，主要有：

  • JavaScript&emsp;&emsp;通过用户或其他与浏览器相关事件捕获交互行为。
  • XMLHttpRequest对象&emsp;&emsp;通过这个对象可以在不中断其它浏览器任务的情况下向服务器发送请求。
  • 服务器上的文件&emsp;&emsp;以 XML、HTML 或 JSON 格式保存文本数据。
  • 其它&emsp;&emsp;JavaScript解释来自服务器的数据(比如 PHP 从 MySQL 获取的数据)并将其呈现到页面上。

Ajax的异步是指不需要进入下一个页面，把数据传到后台去处理。
