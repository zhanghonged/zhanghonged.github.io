发表于 2018-03-01 分类于 Saltstack

&nbsp;&nbsp;&nbsp;&nbsp;Saltstatck提供了丰富的功能模块，涉及操作系统的基础功能，常用工具支持等。
通过官网查看更多详细信息 https://docs.saltstack.com/en/latest/ref/modules/all/index.html
也可以通过 sys 模块列出当前版本支持的功能模块。
salt ‘*’ sys.list_modules

其他模块

除了上述模块外，saltstack还提供了user(系统用户模块）、group（系统组模块）、partition（系统分区模块）、puppet（puppet管理模块）、system（系统重启、关机模块）、timezone（时区管理模块）、nginx（nginx管理模块）、mount（文件系统挂载模块）等等。当然我们也可以通过Python扩展模块来满足需求。
