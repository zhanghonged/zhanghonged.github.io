DRF-Serializers

发表于 2018-04-25 | 分类于 Django REST framework

Serializers 可以将复杂的数据结构变成json或者xml格式，还有反序列化功能，在传入数据校验成功后将解析后数据转换为复杂的数据结构。

serializers有以下几个作用：

  • 将queryset与model实例等进行序列化，转化成json格式，返回给用户(api接口)。
  • 将post与patch/put的上来的数据进行验证。
  • 对post与patch/put数据进行处理。

二、ModelSerializer

大多数情况下，我们都是基于model字段去开发。因此强烈建议使用 ModelSerializer 进行序列化。

1、创建序列化类

ModelSerializer提供一种快捷的方法自动创建与模型字段相对应的的 Serializer class.
ModelSerializer已经重载了 create 与 update 方法，它能够满足将 post 或 patch 上来的数据进行进行直接地创建与更新，除非有额外需求，那么就可以重载create与update方法。
ModelSerializer在Meta中设置fields字段，系统会自动进行映射，省去每个字段再写一个field。
