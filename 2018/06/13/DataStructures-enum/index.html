发表于 2018-06-13 | 分类于 PyMOTW-3

enum：枚举类型。
enum 模块定义了具有迭代和比较功能的枚举类型，使用预先定义好的值来代替数值和字符串。

2、迭代

遍历 enum 类将生成枚举的各个成员。

成员按类定义中声明的顺序生成，names 和 values 不能用于以任何方式对它们进行排序。

3、对比枚举值

由于枚举成员并不是有序的，因此只能通过身份进行对比

枚举值之间的顺序比较是不受支持的，sort相关的大于小于比较运算符会引发 TypeError 异常。

对于绝大多数代码，强烈建议使用 Enum，因为 IntEnum 打破了枚举的某些语义承诺（通过与整数相比较，并因此与其他无关的枚举）。它应该仅在没有其他选择的特殊情况下使用；例如，当整数常量被枚举替代，并且向后兼容性需要仍然期望整数的代码。

5、unique唯一枚举值

enum.unique() 枚举类装饰器，确保只有一个名称绑定到任何一个值。
默认情况下，枚举允许多个名称作为同一值的别名，并且别名不会在 Enum 迭代器中触发重复的值。

由于by_design和closed是其他成员的别名，在遍历 enum 的时候他们并不会输出。成员的规范名称是这个值得第一个名称。

6、以编程方式创建枚举

在某些情况下，以编程方式创建枚举相比于在类定义中硬编码它们更方便。
对于这些情况，Enum还支持将成员名称和值传递给类构造函数。

value 参数是枚举定义的名称，在构建成员的时候使用。names 参数列出了枚举的成员。
当一个单独字符串参数被传递过来时，它会通过空格或逗号进行分割得到的成员的名称，这些成员的值自动分配，并从1开始。
