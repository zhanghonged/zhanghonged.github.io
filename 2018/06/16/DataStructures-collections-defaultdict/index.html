DataStructures-collections-defaultdict

发表于 2018-06-16 | 分类于 PyMOTW-3

defaultdict：为不存在的key提供一个默认value。

class collections.defaultdict([default_factory[, …]])
返回一个新的类似字典的对象。defaultdict 是内置 dict 类的子类。它覆盖一个方法，并添加一个可写的实例变量。其余的功能与 dict 类相同。
第一个参数提供 default_factory 属性的初始值；它默认为 None。所有剩余的参数都视为与传递给 dict 构造函数的参数相同，包括关键字参数。
